# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a documentation-only repository containing a study guide for the **Claude Certified Architect – Foundations** certification exam. It is an independent, community-created resource (not affiliated with Anthropic).

There are no build tools, test suites, or runnable code. All work here is authoring and editing Markdown content.

## Content Structure

The repository has two primary files:

- **`README.md`** — overview, audience description, topic list, and license info
- **`exam-preparation-guide.md`** — the full study guide (the main deliverable)

The guide covers seven sections: tool interface design, error handling, conversation context management, system prompt engineering, MCP, agentic patterns, and agentic workflow design. Each section follows a consistent structure: *What to Know*, *Key Relationships*, *Common Pitfalls*, and *Go Deeper*. The guide closes with a Study Strategy, Quick Reference Cheat Sheet, and Recommended Reading.

## Authoring Guidelines

- Maintain the existing section structure (`### What to Know`, `### Key Relationships`, `### Common Pitfalls`, `### Go Deeper`) for consistency across all seven numbered sections.
- The guide targets developers and solution architects new to LLMs — explanations should be self-contained and not assume prior LLM experience.
- Use concrete, scenario-based reasoning rather than abstract rules. Show trade-offs in context.
- The Quick Reference Cheat Sheet at the end should stay in sync with any content changes in the main sections.
- This guide is licensed CC BY 4.0 — keep the disclaimer and license sections accurate if adding to README.md.
